## Introduction

Carrot Market is a web application that allows users to buy and sell items to each other. It is a clone of the popular Korean app, Karrot Market.

## Features

- User authentication
- User profiles
- User location
- Item listings
- Item categories
- Item search
- Item reviews
- Item ratings
- Item favorites
- Item notifications
- Item image upload

## Technologies

- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma
- PlanetScale
